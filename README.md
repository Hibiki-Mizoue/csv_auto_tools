# CSV自動集計ツール

このリポジトリは、CSV データを読み込み、**日付別・商品別の売上集計**を自動で行い、整形された Excel ファイルを生成する Python ツールです。  
生成される Excel ファイルは、以下の特徴があります。

- タイトルは表の上に表示（横幅に応じて自動調整）  
- ヘッダー下に二重線、表全体は太枠  
- 売上列は桁区切りで表示  
- 表は B3 から開始し、1行余白を確保  

---

## 必要条件

- Python 3.8 以上  
- 必要ライブラリ：
  ```bash
  pip install pandas openpyxl
